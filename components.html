Biblioteca de Componentes do projeto Photo Album

Os componentes do projeto tiveram a marcação HTML e a estilização CSS cuidadosamente feitas.

Abaixo seguem os principais componentes, sendo eles:

  1. Álbum (item de álbum)
  2. Lista de Álbuns
  3. Foto (item do painel de fotos)
  4. Painel de fotos
  5. Área de busca
  6. Slide Show
